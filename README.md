# Wifirst CLI portal login tool

`wifirst_login` is a very simple CLI tool to login into the public wifirst 
network that is widely found in student dorms.

Several tools already exist but none are up to date with the new Wifirst portal that has the following link https://portal-front.wifirst.net/.

You can check [the following repo](https://github.com/MarcVillain/wifirst-connect) which demonstrates how to have a service that automatically starts on startup. If you do the following, you should choose where to store your credentials and add a while loop to login whenever the connection is lost.
